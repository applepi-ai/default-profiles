# business_context/

Business context documents that help you understand the organization behind the data.

## Expected structure

- **`README.md`** — Overview of the business (what the company does, main products, key markets)
- **`[divisions]/[division_name]/README.md`** — Per-division context (optional, only if relevant)
- **`[departments]/[department_name]/README.md`** — Per-department or product-line context
